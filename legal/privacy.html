Privacy Policy

Draft — Code-drafted, pre-legal-review. The final version is confirmed before publication. Last updated: 2026-06-23 (draft).

1. What we collect

ISAAC.GG reads Overwatch match data via the Overwolf Game Events Provider (GEP) and stores it locally on your PC (browser IndexedDB):

  • Match metadata (map, mode, result, duration)
  • Kill-feed events — including the in-game names and heroes of other players in your match
  • Your own hero/death/kill stats, focus/graduation progress, and settings
  • (Optional, on by default) automatic death-clip recordings — video files stay on your PC only

2. Storage and transmission

By default everything is stored locally and does not leave your PC. Auto-backups are written to Documents/ISAAC.GG/ on your PC (no external upload).

Exception — Mode B “AI death-review” (sent only when you run it): when you press the button, ISAAC sends the death clip you chose and its match context (your hero, the enemy who eliminated you and the roster hero names, the detected situation, and the coaching text used) to the ISAAC.GG server (Google Cloud Run, asia-northeast1), which forwards it to Google’s Gemini API for analysis.

  • The clip and context are not stored on the server (pass-through only). The generated reading text is stored locally on your PC.
  • This happens only when you select a death and press the button — never automatically, never for every death.
  • The portion sent to Gemini is subject to Google’s API data handling. Ads are served via Overwolf; ad data handling follows Overwolf’s privacy policy.

3. Identity (Free / Pro)

  • No account or sign-in is required for the free tier. To count daily usage we use a resettable device install ID that does not identify you personally.
  • Pro requires signing in to Overwolf only when used, to verify your subscription via the Overwolf session token.
  • To prevent ad abuse we use a short-lived, server-signed ad ticket together with the install ID. Neither contains personal information.
  • When you agree to the terms, we record that agreement (its version and time) on your PC, and — if you are signed in to Overwolf — also on our server linked to your Overwolf account, so we can prove and carry your consent across devices. This record contains no match data.

4. Other players’ data

Other players’ names from the kill feed are used locally to give your match context (e.g. “killed repeatedly by the same enemy”). For a death you run through Mode B, roster hero names may be sent as that reading’s context (see §2). Apart from this, other players’ names are never transmitted or published.

5. Your rights (deletion)

  • In-app Settings → Data → “Delete all data” permanently removes match history, graduations, settings, death clips, and saved readings from this PC (cannot be undone).
  • Uninstalling alone may leave some data; use the in-app deletion for a full wipe. The server stores no clips or context.

6. Third parties & trademarks

ISAAC.GG is not affiliated with or endorsed by Blizzard Entertainment. Overwatch is a trademark of Blizzard Entertainment, Inc.

7. Changes

If we change how data is handled, we will update this policy. Any new feature that transmits data will, like Mode B, be triggered by your explicit action (opt-in).

8. Contact

Operator: ISAAC.GG (trade name). Our legal name and address are disclosed promptly on request. Bug/privacy inquiries: support@isaacgg.com.

プライバシーポリシー

草案 — Code 起草・弁護士確認前。最終版は公開前に確定します。最終更新: 2026-06-23（草案）。

1. 収集する情報

ISAAC.GG は Overwolf Game Events Provider (GEP) 経由で Overwatch の試合データを読み取り、あなたの PC 内（ブラウザの IndexedDB）にローカル保存します。

  • 試合のメタ情報（マップ・モード・勝敗・時間）
  • キルフィード上のイベント（同じ試合の他プレイヤーのゲーム内名・使用ヒーローを含みます）
  • あなた自身のヒーロー・デス・キル等の統計、フォーカス／卒業の進捗、設定
  • （任意・既定オン）デス前後の自動録画クリップ（動画ファイルはあなたの PC 内のみ）

2. 情報の保存場所と送信

既定ではすべてローカル保存で、あなたの PC を出ません。 自動バックアップはあなたの PC の Documents/ISAAC.GG/ に書き出されます（外部送信なし）。

例外 — Mode B「AI デス読み解き」（実行したときだけ送信）： ボタンを押したときに限り、選んだデスクリップとその試合文脈（あなたの使用ヒーロー、あなたを倒した敵とロスター上のヒーロー名、検出された戦況、読み解きに使うコーチング知識のテキスト）を ISAAC.GG のサーバー（Google Cloud Run, asia-northeast1）に送信し、サーバーが Google の Gemini API に転送して読み解きを生成します。

  • クリップ・文脈はサーバーに保存しません（処理のための通過のみ）。生成された読み解きテキストはあなたの PC にローカル保存されます。
  • 送信はあなたがデスを選んでボタンを押したときだけ発生します。自動送信や全デスの送信はしません。
  • Gemini への送信分は Google の API データ取り扱いに従います。広告は Overwolf 経由で表示され、その取り扱いは Overwolf のプライバシーポリシーに従います。

3. 身元の扱い（無料 / Pro）

  • 無料の利用にアカウント登録・ログインは不要です。 1 日の利用回数を数えるため、リセット可能な端末のインストール ID を用います（個人を特定しません）。
  • Pro を使うときのみ Overwolf へのログインが必要で、セッショントークンで購読の有無を確認します。
  • 広告の不正対策として、サーバー署名付きの一時的な広告チケットと上記インストール ID を用います。いずれも個人情報を含みません。
  • 規約に同意すると、その同意（バージョンと日時）をあなたの PC に記録し、Overwolf にログイン済みの場合はあなたの Overwolf アカウントに紐づけてサーバーにも保存します（同意の証跡・端末間の引き継ぎのため）。この記録に試合データは含まれません。

4. 他プレイヤーのデータ

キルフィード由来の他プレイヤー名は、あなたの試合の文脈（例:「同じ敵に倒され続けた」）を示すためにローカルで使用します。Mode B を実行したデスについては、ロスター上のヒーロー名がその読み解きの文脈として送信される場合があります（§2）。これら以外で他プレイヤー名を外部に送信・公開することはありません。

5. あなたの権利（削除）

  • アプリ内の 設定 → データ →「すべてのデータを削除」で、試合履歴・卒業記録・設定・デスクリップ動画・保存済みの読み解き結果をこの PC から完全に削除できます（取り消し不可）。
  • アンインストールだけでは一部データが残る場合があるため、完全削除は上記機能を使ってください。サーバーにはクリップ・文脈を保存していません。

6. 第三者・商標

ISAAC.GG は Blizzard Entertainment と提携・承認関係にありません。Overwatch は Blizzard Entertainment, Inc. の商標です。

7. 変更

データの取り扱いを変える場合は本ポリシーを改定します。新たにサーバー送信を伴う機能を追加するときは、Mode B と同様にあなたの明示的な操作（オプトイン）を起点とします。

8. 問い合わせ

運営者: ISAAC.GG（商号）。正式名称・所在地はご請求があれば遅滞なく開示します。不具合・プライバシーに関するお問い合わせ: support@isaacgg.com。
